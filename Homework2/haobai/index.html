Pokemon Battle Stats Explorer

This dashboard explores Pokemon battle strength in terms of primary type distribution, dual-type combinations, and multi-stat battle profiles. Users can start from the primary type overview, click a type, and then compare how that type appears in dual-type combinations and battle-stat patterns.

Current selection: All Types

View 1: Pokemon Count by Primary Type

Overview/context view. Click a type to highlight it across the dashboard.

View 2: Dual-Type Combination Heatmap

Focus view. Color shows how many Pokemon share each Type 1 and Type 2 combination.

View 3: Multidimensional Battle Stats

Advanced view. Solid lines represent dual-type Pokemon, while dashed lines represent single-type Pokemon.
